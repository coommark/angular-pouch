1. Discuss an algorithm to traverse a tree, depth first.

DEPTH FIRST TREE TRAVERSAL

Answer:

Tree traversal is the process of systematically walking through the tree to visit all the nodes.

Take the image below as our tree, the three (3) methods of Depth-First Traversal we could use to traverse this tree are:

  1. Preorder Algorithm: Visit the root first (print). Do a left-to-right preorder traversal of each subtree.
    Pseudo-Code: 

    Preorder(t)
{
	if t == NULL return
	visit(t)
	Preorder(t->left)
	Preorder(t->right)
}

    Example: A,B,C,D,E,F,G,H,I

  2. Inorder (only makes sense for binary trees) Algorithm: Do an inorder traversal of the left subtree. Visit the root (print). Do an inorder traversal of the right subtree.
    Pseudo-Code: 
    Inorder (t)
{
	if t == NULL return
	Inorder(t->left)
	visit(t)
	Inorder(t->right)
}
    Example: B,C,A,F,E,G,D,I,H

  3. Postorder Algorithm: Do a left-to-right postorder traversal of each subtree. Visit the root (print).
    Pseudo-Code: 

    Postorder(t)
{
	if t == NULL return
	Postorder(t->left)
	Postorder(t->right)
	visit(t)
}
    Example: C,B,F,G,E,I,H,D,A
